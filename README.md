# SCT_CS_Task4 - Keyboard Activity Logger

## Description

This project is a simple Keyboard Activity Logger developed using Python and Tkinter.

It demonstrates:

- GUI development using Tkinter
- Keyboard event handling within the application window
- File handling
- Timestamp generation
- Start and Stop logging
- Clear Log functionality

## Features

- Start Logging
- Stop Logging
- Clear Log
- Save log entries to keystrokes.txt
- Live status indicator

## Technologies Used

- Python
- Tkinter
